FLASK

Первым делом открываем пустую страницу. Для этого используем Flask. Его нужно импортировать и заключить в переменную:

        from flask import Flask
        app = Flask(__name__)

Саму страницу мы открываем следующим образом:

        @app.route('/example (навигационное название страницы)')

В ковычках в скобке пишется ссылка, которая будет видна в строке поиска. Сюда нужно добавить функцию, с помощью которой будет запускаться файл с кодом, пишущий содержимое страницы:

        @app.route('/example (навигационное название страницы)')
        def example():
            return render_template('example.html (Название файла)')

Можно сделать ссылку на другую страницу. Такой код нужно заключить в тег "a" (без кавычек), который нужно закрыть после назввния, финального вида ссылки. Такой код уже помещается уже в код самой страницы.

        <'a' href={url_for('second-example')}>Финальный внешний вид ссылки
